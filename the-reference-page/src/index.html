What is HTML?

Welcome to a short instructional page on how to use HTML. HTML stands for HyperText Markup Language. It was first seen in 1991 when it was first mentioned by Tim Berners-Lee under "HTML Tags". HTML is mainly a markup language that web browsers use to interpet and compose text, audio, images and more.

Here's a peak into the evolution of HTML!

  • HTML 2: First seen on November 24 1995. Tables weren't introduced until May of 1996. Imagine not being able to create a table until a year later. So unorgainized *faints*.

  • HTML 3: We first saw this on January 14 of 1997. One year after the best year most of man-kind has seen. HTML3.2 was first published as a W3C Reccomendation. It was the first version developed adn stadardized by W3C (The World Wide Web Consortium). HTML 3.2 dropped math formulas, and solved issues among various extensions.

  • HTML 4: This one has a lot of significant dates. On December 18th 1997, HTML 4.0 was published by W3C Reccomendation. It was initially code named Cougar. Anatomy of HTML

From the previous photo we can see that the main parts of code are:

  • The Opening Tag: This consists of the name of the element enclosed in angled brackets. This shows us where the element starts

  • The Closing Tag: This is the same as the opening tag, except that it includes a forward slash before the element name. This states where the element ends. I copied this right the article "Getting Started with HTML", so it's not plagiarism, I think. Don't tell the cops!

  • The Content: Basically what's in the element.

  • The Element: The whole package of the opening tag, the closing tag, and the content. Simple like the famous Bahamian cuisine "chicken in the bag". It's literally just chicken in a bag, it's really good!
